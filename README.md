# Sitio Web WeAre Marketing Digital S.A.S.

Sitio web profesional para la agencia digital WeAre Marketing Digital S.A.S. con 4 micrositios principales.

## Estructura del Sitio

El sitio web incluye las siguientes páginas:

1. **index.html** - Página principal
2. **quienes-somos.html** - Sección "Quienes Somos" con espacio para video de presentación
3. **lo-que-hacemos.html** - Sección "Lo que Hacemos" con espacio para video explicativo
4. **nuestros-clientes.html** - Sección "Nuestros Clientes" para documentación de proyectos
5. **blog.html** - Blog para campañas de clientes

## Personalización Pendiente

### 1. Videos

Necesitas agregar los siguientes videos:

#### Video de Presentación (quienes-somos.html)
- **Ubicación**: `assets/video-presentacion.mp4`
- **Formato**: MP4 (también puedes agregar versión WebM)
- **Descripción**: Video de presentación del equipo de WeAre

#### Video de Servicios (lo-que-hacemos.html)
- **Ubicación**: `assets/video-servicios.mp4`
- **Formato**: MP4 (también puedes agregar versión WebM)
- **Descripción**: Video explicativo de los servicios que ofrece WeAre

**Nota**: Los videos deben tener un formato compatible con HTML5. Opcionalmente puedes agregar imágenes de portada (poster) para cada video.

### 2. Imágenes

Necesitas agregar las siguientes imágenes en la carpeta `assets/`:

- `equipo.jpg` - Imagen del equipo (para quienes-somos.html)
- `cliente-1.jpg`, `cliente-2.jpg`, `cliente-3.jpg` - Imágenes de proyectos de clientes
- `testimonial-1.jpg`, `testimonial-2.jpg` - Fotos de clientes para testimonios
- `blog-1.jpg`, `blog-2.jpg`, `blog-3.jpg` - Imágenes para las entradas del blog
- `video-poster-quienes-somos.jpg` - Portada para el video de presentación
- `video-poster-servicios.jpg` - Portada para el video de servicios

### 3. Redes Sociales

Las URLs de las redes sociales están configuradas como placeholders. Actualiza las URLs reales en todos los archivos HTML:

- **Facebook**: Reemplaza `https://www.facebook.com/wearemarketingdigital` con la URL real
- **Instagram**: `https://www.instagram.com/weareaer?igsh=eHY5azQ3aGppbTNv&utm_source=qr` (@weareaer)

Puedes buscar y reemplazar estas URLs en todos los archivos HTML.

### 4. Contenido de Proyectos

En la página `nuestros-clientes.html`, actualiza:
- Nombres de clientes
- Descripciones de proyectos
- Métricas y resultados
- Enlaces a documentación detallada

### 5. Entradas del Blog

En la página `blog.html`, actualiza:
- Títulos y descripciones de campañas
- Detalles específicos de cada campaña
- Fechas de publicación
- Imágenes relacionadas

## Cómo Usar

1. Abre `index.html` en tu navegador para ver el sitio
2. Todos los archivos están listos para usar, solo necesitas agregar los videos e imágenes
3. El sitio es completamente responsivo y se adapta a dispositivos móviles
